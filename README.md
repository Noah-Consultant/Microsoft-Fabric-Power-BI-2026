# Offre Microsoft Fabric & Power BI — PME/ETI Nouvelle-Aquitaine

et les offres proposées par Noah Consultant pour les PME/ETI viticoles et agroalimentaires
de Nouvelle-Aquitaine.

## Fichiers

- `index.html` : page d'offre statique, responsive, mobile-first.
- `rapport_fabric_pme_2026.pdf` (optionnel) : rapport complet 3 000+ mots.
  - Si tu ne l'utilises pas, supprime simplement le lien dans le CTA.

## Déploiement sur GitHub Pages

1. Crée un dépôt GitHub public, par exemple `fabric-offre-na-2026`.
2. Ajoute les fichiers :
   - `index.html`
   - `rapport_fabric_pme_2026.pdf` (si disponible)
   - `README.md`
3. Pousse le tout sur la branche `main`.

Dans GitHub :

1. Va dans **Settings** → **Pages**.
2. Dans **Source**, choisis :
   - `Deploy from a branch`
   - Branch : `main`
   - Folder : `/ (root)`
3. Clique sur **Save**.

Après quelques secondes, ton site sera disponible à une URL du type :

`https://<ton-compte>.github.io/fabric-offre-na-2026/`

La page est entièrement statique (HTML/CSS), sans JavaScript ni backend, et peut être
utilisée directement comme landing page d'offre Fabric & Power BI pour tes campagnes
LinkedIn ou tes emails de prospection.
